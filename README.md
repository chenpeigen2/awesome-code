# awesome-code

A multi-language repository containing various projects and examples across different programming languages and frameworks.

## Repository Structure

- **Go Projects**: Fyne GUI applications, algorithm implementations, and system tools
- **Python Projects**: Various Python scripts, machine learning exercises, and utilities  
- **TypeScript/Bun**: Modern TypeScript projects using Bun runtime
- **C++ Projects**: C++ reference implementations and examples
- **JavaScript**: Frontend components and utility projects
- **AI**: Machine learning and AI experimentation projects

## Quick Start

### Go Projects
```bash
# Build and run Fyne applications
cd go/myapp/demo
go run ./main.go

# Install Fyne tools
go install fyne.io/fyne/v2/cmd/fyne_demo@latest
```

### Python Projects
```bash
# Install dependencies
cd python
pip install -e .

# Run examples
python main.py
```

### TypeScript/Bun Projects
```bash
cd explore/bun/awesome-bun
bun install
bun run index.ts
```

## Key Features
