# 四天编程学习计划 - 打卡统计

| 日期 | 上午学习内容 | 下午学习内容 | 学习目标 | 完成状态 |
|------|------------|------------|---------|---------|
| **第一天** | 普通数组、二分查找、子串处理 | 滑动窗口、矩阵操作 | 建立基础数据结构认知，掌握线性结构（数组、字符串）的核心操作 | ☐ |
| **第二天** | 链表、堆栈（Stack） | 哈希表、双指针技巧 | 掌握非线性数据结构（链表、堆栈、哈希），熟悉常用算法技巧 | ☐ |
| **第三天** | 二叉树、二叉搜索树 | 图论基础、图论java | 理解树和图的结构，掌握遍历与基础操作，为复杂问题打基础 | ☐ |
| **第四天** | 贪心算法、回溯算法 | 动态规划、技巧总结 | 掌握高级算法思想，提升解决复杂问题的能力，形成知识体系 | ☐ |

## 学习建议：
1. 每日实践：每学完一个主题，用代码实现1-2道经典题目（如LeetCode对应题型）
2. 复习巩固：每天结束前用30分钟回顾当天内容，整理笔记
3. 灵活调整：若某部分掌握较快，可提前进入下一模块；若遇到难点，适当增加练习时间
