## 调用流程图

```
用户交互                     前端 (JavaScript)                    后端 (Rust)
    │                              │                                   │
    │  1. 点击导航链接              │                                   │
    ├─────────────────────────────►│                                   │
    │                              │                                   │
    │                              │  2. 获取 demo 配置                 │
    │                              │     demoConfig[demo]              │
    │                              │                                   │
    │                              │  3. 调用 Tauri API                 │
    │                              ├──────────────────────────────────►│
    │                              │                                   │
    │                              │     invoke('demo_variables')      │
    │                              │                                   │
    │                              │                     4. 路由到对应函数
    │                              │                        level1_basics::
    │                              │                        demo_variables()
    │                              │                                   │
    │                              │                     5. 执行 Rust 代码
    │                              │                        返回数据结构
    │                              │                                   │
    │                              │  6. 返回 JSON 数据                 │
    │                              │◄──────────────────────────────────┤
    │                              │                                   │
    │                              │  7. 渲染到 DOM                     │
    │                              │     renderDemoContent(data)       │
    │                              │                                   │
    │  8. 显示结果                  │                                   │
    │◄─────────────────────────────┤                                   │
    │                              │                                   │
```

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 前端 | HTML5 | 页面结构 |
| 前端 | CSS3 | 样式、暗色主题 |
| 前端 | JavaScript (ES6+) | 交互逻辑、DOM 操作 |
| 前端 | Tauri API | IPC 通信 |
| 桥接 | WebView | 渲染前端 |
| 桥接 | IPC | 进程间通信 |
| 后端 | Rust | 核心逻辑 |
| 后端 | Tauri 2.0 | 桌面应用框架 |
| 后端 | serde | 序列化 |
| 后端 | tokio | 异步运行时 |
