## 🎯 项目概述

这是一个基于 Capacitor 框架的跨平台 Android 应用项目，使用 Web 技术（HTML/CSS/JavaScript）开发，通过 Capacitor 桥接技术访问原生 Android 功能。

## 📱 技术栈概览

```
前端技术:
├── HTML5 + CSS3 + JavaScript
├── 响应式设计
└── 现代 Web API

跨平台框架:
├── Capacitor 5.0
├── WebView 容器技术
└── 插件生态系统

Android 原生:
├── Java 17
├── Android SDK 32
├── Gradle 7.6.4
└── WebView 组件
```
